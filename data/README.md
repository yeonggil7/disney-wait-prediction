# 🏰 ディズニーランド待ち時間データ コレクション

このフォルダには、yosocal.comから取得したディズニーランドの待ち時間データが保存されています。

## 📊 データ概要

- **取得期間**: 2025年6月1日 ～ 7月4日（34日間）
- **総レコード数**: 40,936レコード
- **アトラクション数**: 43個
- **時間スロット**: 28個（8:15 ～ 21:45、30分間隔）

## 📁 ファイル構成

### 🗓️ 日別データファイル
```
yosocal_daily_YYYY-MM-DD.csv
```
- 各日のタイムスタンプ付き待ち時間データ
- 1日あたり約1,204レコード（43アトラクション × 28時間スロット）
- ファイルサイズ: 約95KB/日

### 📋 月間統合ファイル
```
yosocal_monthly_YYYY_MM_timestamp.csv
```
- 全期間の統合データ
- 40,936レコード（3.1MB）
- 全ての日別データを時系列で統合

### 📈 統計情報ファイル
```
yosocal_monthly_stats_YYYY_MM_timestamp.json
```
- 月間データの統計サマリー（JSON形式）
- 人気アトラクション、時間帯パターンなど

### 📄 分析レポート
```
yosocal_analysis_report_timestamp.txt
```
- 詳細な分析結果レポート（テキスト形式）
- 人気ランキング、おすすめ時間、曜日パターンなど

## 📊 データ構造（CSV）

| カラム名 | 説明 | 例 |
|---------|------|-----|
| `time` | 時間スロット | "09:15" |
| `attraction_index` | アトラクション番号 | 35 |
| `attraction_name` | アトラクション名 | "美女と野獣の物語" |
| `wait_time` | 待ち時間（分） | 52 |
| `css_class` | CSSクラス | "B3" |
| `raw_value` | 元データ値 | "52" |
| `date` | 日付 | "2025-06-01" |
| `scraped_at` | 取得日時 | "2025-07-04 21:49:41" |

## 🛠️ データ活用方法

### Python での基本的な読み込み
```python
import pandas as pd

# 月間統合データを読み込み
df = pd.read_csv('yosocal_monthly_2025_06_20250704_215504.csv')

# 特定日のデータを読み込み
daily_df = pd.read_csv('yosocal_daily_2025-06-01.csv')

# 人気アトラクションの分析
popular = df.groupby('attraction_name')['wait_time'].mean().sort_values(ascending=False)
print(popular.head(10))
```

### 分析ツールの使用
```bash
# 基本分析実行
```
