# 🏰 ディズニーシー 待ち時間データ

## 📋 概要
このフォルダには、[yosocal.com](https://yosocal.com/realtime.htm)から取得した**東京ディズニーシー**の待ち時間データが格納されています。

## 📁 ファイル構成

### 📄 日別データ
```
disneysea_daily_YYYY-MM-DD.csv
```
- 各日付の詳細待ち時間データ
- 28時間スロット × 42アトラクション = 1,176レコード/日
- 時間帯別の待ち時間変化を確認可能

### 📊 月間統合データ
```
disneysea_monthly_YYYY_MM_timestamp.csv
```
- 月全体の統合データ
- 全日付のデータを1つのファイルに統合
- 長期的な傾向分析に使用

### 📈 統計情報
```
disneysea_monthly_stats_YYYY_MM_timestamp.json
```
- 月間統計情報（JSON形式）
- 人気アトラクション、平均待ち時間、成功率など

### 📄 詳細レポート
```
disneysea_monthly_report_YYYY_MM_timestamp.txt
```
- 人間が読みやすい形式の詳細レポート
- 人気アトラクションランキング、推奨時間帯など

## 📊 データ形式

### CSVファイル構造
```csv
time,attraction_index,attraction_name,wait_time,css_class,raw_value,date,scraped_at
08:15,1,ビッグバンドビート,0,B8,-,2025-06-01,2025-07-06 10:31:55
09:15,2,タワー・オブ・テラー,5,B0,5,2025-06-01,2025-07-06 10:31:55
```

### フィールド説明
- **time**: 時間帯（8:15-21:45の28スロット）
- **attraction_index**: アトラクション番号（1-42）
- **attraction_name**: アトラクション名
- **wait_time**: 待ち時間（分）
- **css_class**: CSSクラス（B0-B8）
- **raw_value**: 元の値
- **date**: 日付
- **scraped_at**: データ取得日時

## 🔧 使用方法

### 1. 新しい月のデータを取得
```bash
# 2025年7月のデータを取得
python3 yosocal_disneysea_custom_scraper.py --year 2025 --month 7

# 詳細ログ付きで取得
python3 yosocal_disneysea_custom_scraper.py --year 2025 --month 7 --verbose
```

### 2. データ分析
```bash
# 月間データの分析
python3 yosocal_data_analyzer.py --data_dir Disneysea

# 人気アトラクションランキング
python3 yosocal_data_analyzer.py --data_dir Disneysea --analysis popularity
```

### 3. Python での読み込み
```python
import pandas as pd

# 日別データの読み込み
df_daily = pd.read_csv('Disneysea/disneysea_daily_2025-06-01.csv')

# 月間データの読み込み
df_monthly = pd.read_csv('Disneysea/disneysea_monthly_2025_06_timestamp.csv')
```

## ⏰ 推奨時間帯

### 🌅 空いている時間帯
- **8:15-9:00**: 開園直後（ほぼ全てのアトラクションが空き）
- **21:00-21:45**: 閉園前（一部アトラクションが空き）

### 🏃‍♂️ 混雑時間帯
- **10:00-16:00**: 午前中から午後の混雑ピーク
- **土日祝日**: 平日より全般的に混雑

## 🎯 攻略のポイント

1. **🌅 開園ダッシュ**: 人気アトラクションは開園直後が狙い目
2. **📱 リアルタイム確認**: 待ち時間は時間帯により大幅に変動
3. **🎢 分散戦略**: 人気アトラクションと空いているアトラクションを組み合わせ
4. **🍽️ 食事時間**: 12:00-13:00、18:00-19:00は比較的空く傾向
